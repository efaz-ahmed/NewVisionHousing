# New Vision Housing — Website V1

Single-page marketing + investor-relations site for **New Vision Housing Ltd (NVH)**, a UK private limited company offering ethical, share-based, debt-free property investment grounded in Islamic finance principles.

## Live file

`IndexV34.html` — self-contained HTML (~10 MB) with all images, fonts, and assets inlined via a custom bundler. No external dependencies at runtime.

## Sections

1. **Hero** — brand promise, investor CTA, stat counters (shareholders / AUM / properties)
2. **Belief** — ethical framework grounded in Shariah-compliant investing
3. **Journey** — pinned horizontal-scroll timeline of NVH milestones from 2017 onward
4. **Portfolio** — core holdings (residential + commercial UK property) + sister-company ventures
5. **Vision 10** — upcoming flagship venture (Matrimony Ltd) and roadmap
6. **Benefits** — shareholder benefits including Hajj Pot, Wedding Pot, AGM participation
7. **Invest** — pathway, partnership structure, and contact form (`#get-in-touch`)

## Build pipeline

The site is generated by a chain of Python build scripts that progressively patch the source HTML. The current head is `build_v34.py` which consolidates seven layered patch scripts into a single mobile-only style block plus a MutationObserver-based map renderer.

```bash
python build_v34.py    # reads IndexV33.html, writes IndexV34.html
```

## Mobile responsiveness

Breakpoints used:
- `≤1080px` — drawer takes over from top-nav
- `≤900px` — portfolio core-holdings card switches to vertical stack
- `≤768px` — hero BG fix, flagship image hidden
- `≤640px` — hero typography centred + bold + 35% larger; stats become single row
